# Mahadev 555 Store

Mahadev 555 Store is a modern fashion website built using HTML, CSS, and JavaScript. It showcases dresses for men, women, and kids with a clean UI and smooth animations.

## Features

- Animated background video
- Product gallery section
- Category filters (Men, Women, Kids)
- WhatsApp order integration
- Responsive design (mobile + desktop)
- Smooth hover effects

## Live Website

https://phenomenal-pithivier-a07cc5.netlify.app/

## Technologies Used

- HTML
- CSS
- JavaScript

## How It Works

- Users browse products on homepage
- Click “Gallery” to view all collections
- Click “Order Now” to open WhatsApp with product message

## Project Structure

- index.html → main website file
- Images → online links used for products
- No backend used
